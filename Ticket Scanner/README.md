# pdf-ticket-scanner

## Project setup
```
npm install
npm install -g @vue/cli
npm install jspdf --save

```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Deploy to Firebase Hosting
```
cd to directory
c:\$ firebase login
c:\$ firebase init
select hosting from console
Select project from console
Add dist as the public directory- Enter dist in console
configure to single page app- Enter Y in console
npm run build
c:\$ firebase deploy
```
